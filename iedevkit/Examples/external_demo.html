Window.External Demo
Window.External Javascript function allows a pages scripts to access objects and functions from the webbrowsers parent makes available. In our code, you can see clsExtender_GetExternal returns a reference to its main form. This gives the html scripting access to all of that forms objects and functions.

Below are 3 buttons, the first read the current url from the navigation textbox. The second, calls a function on the main form and receives an object reference to the textbox. The 3rd button, will call a function on the main form passing in a value and receiving back a modified value.
