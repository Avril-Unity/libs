clsIeIntegration


   

This class gives you easy access to IE integration tools that allow you to add your own custom icons to the toolbars, and custom entries to the right click menus.
 
The right click menu entries are context ensitive and allowing you to specify for what types of objects menus they will show on. (Images show one set of right click menu options, links another etc)

Enum MenuShowWhen

This enumeration holds all of the possible menu types defined and allows you to easily choose when your new menu item will appear. All of the enumeration names are intutive and define what action they are relating to.

Sub AddToRightClickMenu( _
               menuText As String, _
               sFilePath As String, _
               Optional showWhen As MenuShowWhen = swDefault, _
               Optional showWindow As Boolean = False)


This sub will allow you to create you own menu item. Menu text is the text to appear on the menu, sFilepath is the physical path to the file you wish to run when the user selects your menu item. It must be a html file. This method of integration will execute the script contained in the file within the context of the parent page (or frame of parent page if that is where click initated)

    ShowWhen allows you to define what right click menu your item will appear on. It is defined from the ShowWhen enumeration.

    If the argument showWindow is true, then Internet explorer will create a html dialog window for your script and it can present its own UI. If false, the script simply runs in without a window.

    Scripts run in this manner are given full security access meaning that you can use the Scripting.FileSystemObject and other controls that are not specifically  marked as safe for scripting.

Function RemoveFromRightClickMenu(menuText As String) As Boolean

this function will remove the specified menu item. The argument menuText is the text string of the menu item. Returns false if the item was not found.
 
Function ShowInstalledMenuExtensions() As String()

Enumerates all of the menu items added to IE in this manner. Returns a string array of menu names.
 
Function IntegrateWithIEToolbar( buttonText As String, _
                                                    hotIconPath As String, _
                                                    defaultIconPath As String, _
                                                    exeOrScriptPath As String, _
                                                    Optional isVisible As Boolean = True _
) As String


This function allows you to easily add your own custom icons to the IE toolbar

buttonText is the tooltip to show
hoticonPath is the mouseover icon
defaulticonpath is the standard display icon 

exeorscriptpath is the path to the file you want executed when the user presses the button. If it is a html file, it will be executed with no parent window as a menu item would. If it is an executable it will be marked with the Exec flag. 

isVisible marks whether the icon appears by default on the toolbar tab.

    Note: If the user has customized the tool at all, it will not become visible by default unless they reset thier toolbar or manually add it. I am researching how to get around this.

 
This function returns a text string that is your key to its entry. If you ever plan on modifying or removing entries added you should save this return value somewhere.


Function RemoveIEToolBar(sGuid As String) As Boolean

sGuid is the key returned from the Integrate funciton. Returns false if entry is not found.
 
Function ModifyToolBarSetting(sGuid As String, _
                                                   Optional buttonText As String, _
                                                   Optional hotIconPath As String, _
                                                   Optional defaultIconPath As String, _
                                                   Optional exeOrScriptPath As String, _
                                                   Optional isVisible As Boolean = True)

This function allows you modify portions of the toolbar configuration.
 
All of the arguments are the same as in the Integrate function.
 
sGuid is the key passed back from Integrate.
