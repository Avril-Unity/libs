IE Developers Pack


A very frequently used component for the VB developer is the Internet Explorer control. It lends itself to all kinds of functions because of its adaptability, network enablement, and powerful document rendering capabilities.

It does however lack in certain areas for the Vb developer. How often have you wanted:
  • remove its 3d borders
  • turn off its scrollbars
  • ensure that scripting is enabled
  • replace or turn off standard context menus
  • add buttons to the IE toolbar
  • add your own items to IE context sensitive menus
  • access VB form objects and functions from javascript
With the IE Developers Kit, all of this is now easily within your grasp, and these are just a handful of the things that the IE developers pack can help you do.

All of these things have previously taken alot of code and knowledge to pull off. This has effectivly left the majority of users having to just accept what they had to live without. Just think of the possibilities these new features open and you will be amazed.

This component encapsulates literally months of research, reading, and experimentation. As with all other SandSprite.com components, the clean interface design and user friendly functions, will make this a breeze to use for developers of any skill level.
