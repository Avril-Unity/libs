clsSecurityManager


   

The Security Manager class allows you to set the global Internet Explorer options for the host machine. You can set  any of the options available in the standard IE options dialog and for any security zone.


Enum ZoneTemplates

enumeration of the templates you can set, these are the low, medium and high zones you see on the inital IE options page.

Enum urlZone

enumeration of all of the security zones you can manipulate
 
Enum urlActions

this is an enumeration of a host of the options in the IE set custom panel.

Sub ShowInternetOptions()

shows the standard IE options dialog
 
Sub SetPolicyToTemlpate( template As ZoneTemplates,
                                           Optional Zone As urlZone = zInternet)

allows you to set a security zone to a default machine template
 
Function IsPolicyActive( p As urlActions,
                                        Optional Zone As urlZone = zInternet _ < BR > ) As Boolean

Allows you to check to see if a certain feature is enabled for the security zone you are intrested in. This is how you could determine if javascript is currently enabled, or if safe for scripting checks were active.

Sub SetPolicy(  p As urlActions, _
                         active As Boolean , _ 
                         Optional Zone As urlZone = zInternet)


this sub allows you to set the value for the security zone you choose. Here is where you could turn scripts on for you clients etc...
