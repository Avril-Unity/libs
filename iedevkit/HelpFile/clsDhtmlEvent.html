clsDhtmlEvent


   

    This is a light class thrown in because it has definate times when it can come to the rescue.

    This class will allow you to attach VB functions to Web page events through html objects event handlers.

    For example, you can make a html button in a web page directly call a function in your VB code. 

   With this method, this can be done without having to expose your entire forms objects and classes to manipulation through the GetExternal Method.

 

Sub SetReference(  parent As Object, _ 
                               sFunction As String, _ 
                               Optional CallingArgument As String)

parent is an object reference to the form or class that holds the function you are attaching.

sFunction, is the exact string name of the function you wish called when the event fires.

 
CallingArgument is the argument to be passed to the function.

See the example code for the exact usage of this class and technique.

Function DefaultFunction()

This is the function that will be called by the webpage event. You do not call this function manually unless you want to test the reference you initally set. See the code example for how to set this class up to use it .
