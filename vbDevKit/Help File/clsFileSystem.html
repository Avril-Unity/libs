clsFileSystem

    No mater what type of program you are creating, it is very often necessary to have to work with the filesystem. The built in VB Filesystem commands do not have intellisense available and often leave you running for a book or double checkign syntax even if you have been using it for years. This ineviatable leads to bugs and slow developement time.

    The Vb Developers Kit clsFileSystem wraps all of the file system manipulation functions in a very intutive and easy to use library that will not only cut your developement time, but also your debugging time. This library has been built up over the course of several years, and has everythign you could need for general everyday file writing.

Function GetFolderFiles(folderPath As String, Optional filter As String = "*", Optional retFullPath As Boolean = True) As String()

This function will return a string array of the files in the folder you specify. The filter argument allows you to define just the file types you want returned, like *.pdf or *.htm* The retFullPath argument defines wether you want the file paths returned to be absolute full paths ( c:\folder\file.htm ) or just the file name.

Function GetSubFolders(folderPath As String, Optional retFullPath As Boolean = True) As String()

This function will return a string array of all of the subfolders contained in the parent folder. RetFullpath defines wether it is a full path returned or just the sub folder name.

Function FolderExists(path As String) As Boolean

Returns a boolean if the specified folder exists or not

Function FileExists(path As String) As Boolean

Returns a boolean if the specified file exists or not

Function GetParentFolder(path) As String

Returns a string containing the full path to the parent folder of the path you pass in.

Sub CreateFolder(path As String)

Creates folder specified. Be sure parent folder exists if several layers deep, or see BuildPath function which will build all folders to final path.

Function FileNameFromPath(fullpath As String) As String

Returns just the name of a file given its full path

Function WebFileNameFromPath(fullpath As String)

Same as above but used for web addresses.

Function DeleteFile(fpath As String) As Boolean

Tries to delete teh specified file. Returns true if successful.

Sub Rename(fullpath As String, newName As String)

Renames specified file. first argument is the full path to teh file. Second argument is the new file name desired (no path included)

Sub SetAttribute(fpath, it As VbFileAttribute)

Sets the file system attribute for the file given the full path.

Function GetExtension(path) As String

Returns just the file extension for the full path. ex: path="c:\boot.ini" returns ".ini"

Function GetBaseName(path As String) As String

Returns just the name of the file, ommitting the extension.

Function ChangeExt(path As String, ext As String)

Changes teh extension of the file. Path is the full path to the file, ext is the new extension. If path is just a file name and not the path to a file, then the entension of that string filename will be changed.

Function SafeFileName(proposed As String) As String

The file system has some characters that cannot be present in file names. This function will remove these characters and will return a safe file name.

Function RandomNum() As Long

Returns a random number, used internally to generate freefile names but exposed publically for utility.

Function GetFreeFileName(folder As String, Optional extension = ".txt") As String

Given the path to a folder it will generate a numeric filename that is guaranteed to be free in that folder. Ext signifies the extension you want for the file.

Function buildPath(folderPath As String) As Boolean

This function will try to create the fill folder path you specify even if some of the parent folders do not exist. Returns true if successful.

Function readFile(filename) As Variant

Reads the entire contents of a file and returns it. Retursn variant making this useful for binary data as well. Makes no attempts to verify file exists, if not throws error.

Sub writeFile(path As String, it As Variant)

Writes the file specified with the data passed in in variable it. It is variant and also works with binary data.

Sub AppendFile(path, it)

Appends file with specified data.

Function Copy(fpath As String, toFolder As String)

Copies the file specified to the new folder using the same file name.

Function Move(fpath As String, toFolder As String)

Moves teh file specified to the new folder.

Sub CreateFile(fpath As String)

Creates a 0 byte file.

Function DeleteFolder(folderPath As String, Optional force As Boolean = True) As Boolean

Deletes Specified folder. If force parameter is true, it will attempt to recursivly delete all sub file s and folders. if successful returns true. (Can fail if files are open or locked in path)

Function FolderName(folderPath) As String

Returns just the name of the last folder given a path to a specified folder.
