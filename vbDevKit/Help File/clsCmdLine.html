clsCmdLine


    This class gives you easy access to the command line that your program was started with. (Say if from DOS your program was started with ' /b -s "file name" ' that would constitute 3 arguments /b, s, and "file name"

    There are many methods of parsing commandlines and passing in variables. This implementation defines any argument precedeed by a / to be a single argument up to teh first space. Thus /bat is all one argument. Arguments preceeded by a - are taken as multiple arguments. Thus -bat would be parsed as 3 arguments b,a,t . An argument with no prefix such as

myprog.exe bat ball

    Is taken as 2 seperate agruments bat, and ball. If the argument must contain spaces both single and double quoted strings are recgonized:

myprog.exe "c:\my file.txt" 'c:\file 2.txt'

    Redirection operators are not supported. They are a function of the operating system and your specific code. They are not related to the command line passed into the program.

Property Get CommandLine() As String

Returns the Command line your program was started with.

Property Let CommandLine(cmd As String)

Allows you to set the command line to be parsed. This is useful if you allow your users a commandline type interface to some interactive functionality.

Function IsArgPresent(argName As String) As Boolean

Returns true if an argument is present. ex: If you wanted to check if they used the /bat option this is an easy way to test for its presence.

Function GetArgumentsToArray() As String()

Returns a string array of all of the arguments in the commandline.
