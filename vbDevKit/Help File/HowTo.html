How To


    The first step in using the VB Developers Kit in your own project is to add a reference to it in your project properties.

From VB 6 goto Project -> References menu



Scan down the list until you see "VB Developers Kit from SandSprite.com"



Put a check in the box and hit ok. Now when are in a code window, you should see a dialog similar to the following.



    The libraries should be intutive enough and easy to follow from thier function names and arguments. If not, Refer to the classes help page and you will be up and coding in no time flat !

    Also be sure to check out the example code in the Examples Folder.
