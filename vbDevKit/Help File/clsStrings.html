clsStrings

    This class is thrown in as a bonus. It provides many utilities to make your string parsing life much easier. It is built around a loose Javascript Object model. You set teh string you are parsing once instead of having constantly keep including it in function calls. Also the parsing has been augmented with the ability to set and manipulate a pointer in the string. This allows you do some really neat things, and very effectivly.

    If you ever have to do complex string parsing, I guarantee this class will save you ALOT of typing and ALOT of bugs. I use it every time i am in a stirng parsing nightmare...and it makes it easy and bearable.

Public Function IndexOf(it, Optional startAt = 1, Optional advPtrLenIt = False)

Returns the position of the substring "it" in the string to be parsed, optionally starting at the position you provide. If AdvPtrLenIt is true, then the pointer character is set to end of the string found. If false, pointer is still advanced to the IndexOf position for use in subsequent functions. This function also sets an internal variable that records the string you tried to locate for use in functions such as NextIndexOf()

Public Function NextIndexOf(Optional ChangePtrChrTo = "")

Searchs for the character you did an inital IndexOf search on, Allows you to change this pointer character for the search. This could be useful for example if you wanted to search for the opening [ you would do an indexOf("[") then to find the matching ] you could NextIndexOf("]") the search would start at the pointer character, which was set by indexOf to the position of the first [

Sub SetDelimitedValues(strIn As String, delimiter As String, defaultVal, ParamArray FillVariables())

Bonus function, useful in string parsing, but not really related to teh library. strIn is the string to parse, delimiter is the character you want strin broken up at, and the last argument is as many variables as you were expencing to have filled from the string.

For example, if you were expecing a string of 3 comma seperated variables. and they were age,weight,eyecolor you might do the following. 
	dim age,weight,eyeColor,s as string 
	s = "21,150,blue"
	SetDelimitedValues s, "," , "Not Found", age, weight, eyecolor


The default value specifies what value to assign a variable if no value was found in the delimited string for it. This would be used in the case that s="21,150"

Here it will not throw an error, just assign the default value of "Not Found" as specified above. You can use this to set as many variables as you want all in one line. It is a special use function...but it does come in very handy at times.
