clsCmnDlg


    Common Dialog functions are forms that windows provides some standard functionality in. Some frequently used common dialogs are the Save File, Open Folder, Choose Color, and Open File dialogs. Whatever windows program you use, you may have noticed that these dialogs always the same. This is because people usally use the ones provided by Windows.

    In order to use the Windows provided ones, you have to use the Windows API calls, or reference the microsoft provided cmndlg.ocx control. This class wraps the API functions for you in an easy to use format that wont force you to distribute another bulky control with your project.

Property Let/Get ErrorOnCancel(bln As Boolean)

    This property allows you to set your preference wether to return a zero length string if the user presses cancel, or to raise an error. The default is to return a zero length string.

Sub SetCustomFilter(displayText As String, Optional wildCardExtMatch = "*.*")

    Lets you define a custom file type to show in the open file dialog. Once you set this, Just call the OpenDialog function and specify the CustomFilter as the filtertype and this value will be used.

Function OpenDialog(filt As FilterTypes, Optional initDir As String, Optional title As String, Optional pHwnd As Long = 0) As String

    This will display the OpenFile Fialog. the filtertype is define from the filterTypes enumeration that holds a listing of commonly used file types. If you need to use a custom fule type. That too is available using the SetCustomFilter function above. InitDir is the initial directory to start in. Title is the window title, pHwnd is the optional parent window handle usally set to me.hwnd

Function SaveDialog(filt As FilterTypes, Optional initDir As String, Optional title As String = "", Optional ConfirmOvewrite As Boolean = True, Optional pHwnd As Long = 0) As String

    This displays the file Save Dialog, arguments are the same as above.

Function ColorDialog(Optional pHwnd As Long) As Long

    Displays the Choose Color dialog. pHwnd is the optional handle of the parent window usally set to me.hwnd

Function FolderDialog(Optional initDir As String, Optional pHwnd As Long = 0) As String

    Shows the Choose Folder dialog. InitDir allows you set the directory to start in, pHwnd as above.
