clsRegistry


    This class gives you easy access to the registry Functions. The registry is a built in windows database that holds many of the systems configuration settings. The registry is supposed to take the place of the use of INI files. Vb only has built in access to a minute portion of the registry through GetSetting, SaveSetting. This Library will allow you full access to all of the registrys branchs and capabilities.

    Be warned you can really mess up your system if you make delete or change stuff in the registry. Know what you are doing before you attempt to do it in code or you mayfind yourself reinstalling your system. Dont say i didnt warn you.

    This library includes some built in enums that make registry access a snap. the hkey Enum holds all of the branchs of the registry and lets you specify which you want to write to. The dataType enumeration allows you to specify which type of value to write.

Function keyExists(hive As hKey, path) As Boolean

Returns true is the specified path exists.

Function DeleteValue(hive As hKey, path, ValueName) As Boolean

Deletes the specified value. Returns true if successful.

Function DeleteKey(hive As hKey, path) As Boolean

Deletes the specified key. Returns true if successful.

Function CreateKey(hive As hKey, path) As Boolean

Creates the key specified, returns true if successful.

Function SetValue(hive As hKey, path, KeyName, data, dType As dataType) As Boolean

Sets teh specified value, returns true if successful.

Function ReadValue(hive As hKey, path, ByVal KeyName)

Returns the value specified from the hive and path.

Function EnumKeys(hive As hKey, path) As String()

Returns a string array of all of the keys in the specified folder.

Function EnumValues(hive As hKey, path) As String()

Returns a string array of all of the values in the specified path.
